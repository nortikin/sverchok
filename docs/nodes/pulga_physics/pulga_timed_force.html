Pulga Timed Force¶

Functionality¶

This node creates a force to be applied with the Pulga Physics Solver node. The inputted force will only be applied between the defined iterations range.

Input¶

  • Force: Force to apply the time restriction to.

  • Start: Start iteration when the force will be applied.

  • End: End iteration when the force will stop being applied.

Examples¶

Example of description:

  • Generator-> Segment

  • Fields-> Rotation Field

  • Modifiers->Modifier Make-> UV Connection

  • Number-> Number Range

  • List->List Main-> List Join

  • Viz-> Viewer Draw

  • Scene-> Frame Info

  • Pulga Physics-> Pulga Physics Solver

  • Pulga Physics-> Pulga Vector Force

  • Pulga Physics-> Pulga Random Force

In the following example a Random Force will be applied on the first 50 iterations, after a Fit Force will be applied.

  • Generator-> Plane

  • Generator-> IcoSphere

  • List->List Main-> List Join

  • Viz-> Viewer Draw

  • Scene-> Frame Info

  • Pulga Physics-> Pulga Physics Solver

  • Pulga Physics-> Pulga Random Force

  • Pulga Physics-> Pulga Fit Force
