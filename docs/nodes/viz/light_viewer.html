Light viewer¶

This node offers a nodification of the currently implemented lamps in Blender. It should be self explanatory. - you can pass it multiple vectors and it will generate an individual lamp for each - if you want unique colors per lamp, then feed it the output of a Colour Out node (with a list-split before hand)

Category¶

Viz -> Light viewer

Inputs¶

  • Origins - list of matrices where to place generated lights

  • Size - size of for some type of lights

  • Strength - energy of a light

  • Color - color of a light

  • Size X - size along X axis for rectangle and ellipse shapes

  • Size Y - size along Y axis for rectangle and ellipse shapes

  • Spot size - size of a lamp of spot type

  • Spot blend - sharpness of light for spot type

Parameters & Features¶

Param

Description

Collection
