Easing 0..1¶

This node offers a finite list of interpolation functions. We’ve collected the easing functions in a visual document here

https://zeffii.github.io/docs_easing_node/

  • (screenshot, open in new tab to show all)

the original development thread is here: https://github.com/nortikin/sverchok/issues/695

You can see the kind of easing curve because we draw it to the right of the node. Add the node to the tree and connect something useful to the input, and scroll through the list of easing curve types.

  • Number-> Number Range

  • Viz-> Viewer 2D

  • Number-> List Input

Ctrl-Mouse scroll over drop down list will scroll items in the list.
