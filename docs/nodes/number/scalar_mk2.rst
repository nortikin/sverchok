Math_mk2
========

Functionality
-------------

This node can transform incoming values by a selected function (trigonometric,
or other mathematical function, and also provide some useful math costant)

Inputs & Parameters
-------------------

**float** or **int**

+---------------+------------------------+
| **class**     | **type**               |
+---------------+------------------------+
| Trig          | Sine, Cosine,          |
|               | Tangent, Arcsine,      |
|               | Arccosine, Arctangent, |
|               | acosh, asinh, atanh,   |
|               | cosh, sinh, tanh.      |
+---------------+------------------------+
| math          | Squareroot, Negate,    |
|               | Degrees, Radians,      |
|               | Absolute, Ceiling,     |
|               | Round, Round N, Fmod   |
|               | modulo, floor,         |
|               | Exponent, log,         |
|               | log1p,log10,           |
|               | +, -, * , /, //,       |
|               | x-1, x+1, x * 2,       |
|               | x/2, x ** 2, **,       |
|               | min, max, mean, gcd    |
+---------------+------------------------+
| constants     | pi, e, phi, tau.       |
+---------------+------------------------+

Outputs
-------

**float** or **int**
