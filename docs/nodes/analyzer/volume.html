Volume¶

Alias: Volume

Functionality¶

Count Volume of every object. Output list of values

Inputs¶

  • Vers vertices of object(s)

  • Pols polygons of object(s)

Outputs¶

  • Volume, corresponding to count of objects it outputs volumes in list.

Examples¶

  • Generator-> NGon

  • Generator-> IcoSphere

  • Matrix-> Matrix Apply to Mesh

  • Matrix-> Matrix In

  • List->List Struct-> List Levels

  • Viz-> Viewer Draw

  • Text-> Stethoscope

  • Text-> String Tools
