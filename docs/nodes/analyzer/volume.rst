*Alias: Volume*

Functionality
-------------

Count Volume of every object. Output list of values

Inputs
------

- Vers vertices of object(s)
- Pols polygons of object(s)


Outputs
-------

- Volume, corresponding to count of objects it outputs volumes in list.
