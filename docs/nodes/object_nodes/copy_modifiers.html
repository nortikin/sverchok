Copy Modifiers Node¶

Functionality¶

The node performs similar operation to standard Blender Copy Modifiers operation. It takes two set of objects and apply modifiers from one set to another. It is useful for example in case when you want to assign Remesh or Subdivide modifiers to Sverchok objects.

Inputs¶

Object To - Objects to whom to apply modifiers

Object From - Objects from which get modifiers

Outputs¶

Object - Objects with assigned modifiers
