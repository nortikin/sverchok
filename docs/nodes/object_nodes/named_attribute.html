Named Attribute¶

Functionality¶

The node takes a blender object and an attribute name and tries to find attribute with such name if it fails it rises an error. The size and type of attributes remains unchanged by the node.

Category¶

BPY Data -> Set Mesh Attribute

Inputs¶

  • Object - Blender data blocks

  • Name - name of attribute to read

Outputs¶

  • Object - Blender data blocks

  • Attribute - Values of the attriubte

Usage¶

Setting and reading some attribute on a mesh
