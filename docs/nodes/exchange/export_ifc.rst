Export IFC
==========

Functionality
-------------

Exports IFC objects to file. It is simple IFC export without any dependencies as IfcOpensHell for example and awoiding installing sverchok-ifc addon. For quick no categorised proxy brep.

**File Type**: IFC (adds extension if you forgot)

**Base Name**: Base name of the object to export. IFC stores it, can be displayed when importing to other software.
