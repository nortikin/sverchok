Receive From Sorcar¶

Functionality¶

A dedicated node to receive mesh data from Sorcar node-tree (one-sided connection).

Outputs¶

Vertices, Edges, Polygons, VerticesMask, EdgesMask and PolygonsMask. All outputs will be generated. The evaluated mesh from “Send to Sverchok” Sorcar node input (referencing this node) will be generated.
