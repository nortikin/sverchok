Vector Drop¶

Functionality¶

Reverse operation to Matrix apply. If matrix apply adding all transformations to vertices. Than vector drop subtract matrix from vertices.

Inputs¶

  • Vectors - Vectors input to transform

  • Matrixes - Matrix to subtract from vertices

Outputs¶

Vectors - vertices

Examples¶

  • Generator-> IcoSphere

  • Generator-> Suzanne

  • Generator-> Box

  • Generator-> Plane

  • Analyzers-> Component Analyzer

  • Modifiers->Modifier Change-> Polygon Boom

  • Matrix-> Matrix In

  • List->List Main-> List Length

  • Viz-> Viewer Draw

  • Viz-> Viewer 2D
