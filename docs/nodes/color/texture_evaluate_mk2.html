Texture Evaluate¶

This node evaluates a texture at specific coordinates

Inputs & Parameters¶

Parameters

Description

Mapping

Vectors mapping mode. Offers:

UV Coordinates (with domain 0 to 1) and

Object (with domain -1 to 1)

Channel

Color channel to output

Offers: Red, Green, Blue, Hue, Saturation, Value, Alpha, RGB Average,

Luminosity and Color

Use Alpha

Toggle to add alpha channel (Only when Channel is set to Color)

Vertices

Vertices of the mesh to displace

Texture

Texture(s) to use as base

  • Logic-> Logic Functions
