Circle SVG
==========

Functionality
-------------

Creates SVG ellipses (and circles).

Inputs
------

- Center: Folder where de SVG file will be placed.
- Radius X: Horizontal Radius.
- Radius Y: Vertical Radius.
- Angle: Rotation of the ellipse.
- Fill / Stroke: Fill and Stroke attributes.


Outputs
-------

- SVG Objects
