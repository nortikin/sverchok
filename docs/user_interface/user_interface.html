User interface¶

If you are new to Blender node tree interface it is worth to have a look at Blender docs first.

  1. Menu of editor types

  2. Name of the current node tree

  3. Menu of embedded templates in Sverchok. The template will be created in current tree otherwise new tree will be created

  4. Processing shows whether current tree is in Live update mode

  5. Git branch - if you clone and link repository - it will show current branch
