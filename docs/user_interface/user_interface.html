User interface¶

If you are new to Blender node tree interface it is worth to have a look at Blender docs first.

  1. Menu of editor types

  2. Name of the current node tree

  3. Menu of embedded templates in Sverchok. The template will be created in current tree otherwise new tree will be created

  4. Processing shows whether current tree is in Live update mode

  5. Git branch - if you clone and link repository - it will show current branch

There are several ways to create nodes in Sverchok:

  • From “Add Node” menu, which can be invoked from top menu bar, or by pressing Shift-A.

  • From “Sverchok Nodes” toolbox under Tools (T) panel.

  • From standard Blender’s search menu (F3).

  • All panels
    • Property Panel
    • Tool Panel
    • 3D Panel
  • Input socket menus
  • Output socket menus
    • Flatten
    • Simplify
    • Graft
    • Graft Topology
    • Unwrap
    • Wrap
  • Node Shields
  • Shortcuts

Note

Add node menu, Search node menu and Context node menu are not covered currently. Any contributions are welcome.
