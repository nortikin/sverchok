Flip Curve
==========

Functionality
-------------

This node generates a curve by inverting the direction of parametrization of another curve; in other words, it generates the curve identical to provided one, but with the opposite direction.

.. image:: https://user-images.githubusercontent.com/14288520/211383093-d9699de9-d616-4d36-95b9-2d1c2a2fb1f7.png
  :target: https://user-images.githubusercontent.com/14288520/211383093-d9699de9-d616-4d36-95b9-2d1c2a2fb1f7.png

Inputs
------

This node has the following input:

* **Curve**. The curve to be flipped. This input is mandatory.

Outputs
-------

This node has the following output:

* **Curve**. The flipped curve.
