Export GLB
==========

Functionality
-------------

Export file in GLB format and view it in HTML

Inputs
------

* **Vertices** - Vertices
* **Faces** - Polygons

Outputs
-------

- **Export Path** - Information to check correctness
- **Status** - Information to check correctness


Properties
----------

- **View as HTML** - Exports GLB & Opens web browser and look at model
- **Export GLB** - Only export GLB model to path
- **Filename** - File name for model

------

- **Shoiw grid** - HTML View feature
- **Show axes** - HTML View feature
- **Auto rotate** - HTML View feature
- **Background** - HTML View background color
- **Mesh** - HTML View object color
